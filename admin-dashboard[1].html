यह Dashboard कैसे काम करता है:

1. FIR data FIR_Master Google Sheet से live आता है (CSV export URL से)
2. Telegram registrations Telegram_Registrations Google Sheet से आती हैं (Apps Script webhook द्वारा भरी जाती हैं)
3. Alert चैनल column बताता है — अगर IO ने Telegram register किया है तो उसे Telegram पर अलर्ट जाएगा, नहीं तो SMS पर (fallback)
4. इस पूरे dashboard को GitHub Pages पर होस्ट किया गया है ताकि कहीं से भी access किया जा सके

Google Sheet URLs अपडेट करने के लिए: इस फाइल के नीचे स्क्रिप्ट सेक्शन में FIR_SHEET_URL और TELEGRAM_SHEET_URL बदलें।
